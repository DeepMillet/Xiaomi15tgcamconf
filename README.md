# Xiaomi 15T GCam Green Tint Fix

A custom XML configuration file for Google Camera (GCam) on the Xiaomi 15T (MediaTek Dimensity 8400 Ultra). 

This config specifically fixes the green tint issue in RAW photos (often visible when editing in Lightroom) by correcting the Black Level values, while also forcing high-quality HDR+ settings.

## Installation
1. Download the `Xiaomi15T_Fix.xml` file.
2. Open your phone's File Manager and create the following folder path: `Internal Storage > GCam > Configs8` (or `Configs9` depending on your GCam version).
3. Move the `.xml` file into that folder.
4. Open the GCam app and double-tap the black space next to the shutter button.
5. Select `Xiaomi15T_Fix.xml` from the dropdown and tap **Restore**.
